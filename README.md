## The Problem
The screen is the new front door. Anyone walking past can step inside.

- **73%** of professionals report shoulder-surfing in public
- **1 in 4** data breaches involve visual credential exposure
- **$4.9B** estimated annual loss from visual hacking

## Solution
**Privacy Guard** — On-device, real-time, system-wide shield using existing TrueDepth sensor.

**Sense → Detect → Shield → Restore** (all <150ms, Secure Enclave, zero data leaves device)
